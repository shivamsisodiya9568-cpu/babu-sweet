## 🛠️ Customization Guide

### Change Contact Information
Open `index.html` and search for:
```html
<!-- Phone Number -->
+91 98765 43210

<!-- Address -->
Shop No. 12, Main Market, Near City Center, India

<!-- WhatsApp -->
https://wa.me/919876543210
```

### Change Sweet Prices
Search for price values like:
```html
₹280/kg    <!-- Gulab Jamun -->
₹320/kg    <!-- Rasgulla -->
₹680/kg    <!-- Kaju Katli -->
```

### Add Your Shop Photos
1. Save your photos in `public/images/` folder
2. Update image sources in HTML:
```html
<img src="public/images/your-photo.jpg" alt="Shop Photo">
```

### Change Colors
Find and modify the gradient classes:
```css
.sweet-gradient {
    background: linear-gradient(135deg, #f093fb 0%, #f5576c 100%);
}
```

### Update Social Media Links
Search for social icons and update href:
```html
<a href="https://instagram.com/yourshop" class="...">
<a href="https://facebook.com/yourshop" class="...">
```

## 📱 WhatsApp Integration

The website includes a floating WhatsApp button. To change the number:

1. Find this line in `index.html`:
```html
<a href="https://wa.me/919876543210" class="fixed bottom-6 right-6 ...">
```

2. Replace `919876543210` with your WhatsApp number (include country code, no + sign)
